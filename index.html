Hitung Luas Segitiga

Keterangan :
a = alas Segitiga
t = tinggi Segitiga
L = 1/2 x alas x tinggi






Hitung Keliling Segitiga

Keterangan :
A = sisi A Segitiga
B = sisi B Segitiga
C = sisi C Segitiga
K = A + B + C
